### installation

1. clone the repo
    ```bash
    $ git clone https://github.com/MoPaMo/liz.git
    ```
2. install dependencies
   ```bash
    $ npm install 
    ```
3. set your password in the `.env` file
4. run `index.js`
    ```bash
    $ node index.js
    ```
